Core Capabilities

为什么是它

不是又一个机器学习模型，而是一台确定性的数值机器：每一跳都可审计、每一次结果都可复现。

确定性 · 白盒

同输入永远同输出。每一跳的传播向量与剪枝掩码都可逐层检查、可重放——拒绝黑盒，结论可被审计与回测。

微秒级推理

SIMD 加速的稀疏矩阵–向量乘；100 节点 / 5 跳级联 < 100μs。快到可以做交互式「如果……会怎样」与近实时推演。

阈值门控动力学

脆性节点收到信号越过阈值即饱和突破（继电器式雪崩），亚阈值则快速衰减。表达真实「临界点」，超越线性衰减。

固定内存预算

全预分配、无运行时堆增长、无 GC 抖动。可预测的延迟，适合实时与嵌入式部署。

跨平台 SIMD

nightly portable_simd 一份代码，在 aarch64→NEON/SVE、x86_64→AVX2/AVX-512 自动降级，无需运行时硬件检测。

领域无关内核

「实体 + 加权影响边 + 阈值门控」是一套抽象动力学。金融传染、舆情扩散、政策传导，共用同一份引擎代码。

The Engine

数学内核

本质是一句话：加权有向稀疏图上的「线性传播 + 阈值门控 + 幅值剪枝」，有界确定性前向模拟。

cascade.rs — propagation
# 每跳两步：稀疏 SpMV 线性传播，再逐节点门控 + 幅值剪枝
r⁽ᵏ⁾ = A · f⁽ᵏ⁻¹⁾                          # A = CSR 加权耦合矩阵（信号沿 col→row 流动）
f⁽ᵏ⁾ᵢ = r⁽ᵏ⁾ᵢ  if gᵢ(r⁽ᵏ⁾ᵢ, k) ≥ θ  else 0    # 置信度 < θ → 剪枝，停止传播

# 逐节点门控 gᵢ（脆性节点带阈值突破分支）
gᵢ(r,k) = 1.0          若脆性且 r ≥ τᵢ      # 突破：饱和透传（雪崩）
        = r · 0.5ᵏ       若脆性且 r < τᵢ      # 亚阈值：随跳数衰减
        = r · dᵢ          若非脆性             # 线性衰减
输出每节点的「首次命中跳数 / 峰值置信度 / 累积时滞」三张场，外加隐含的突破节点集。一趟在百微秒量级完成。
